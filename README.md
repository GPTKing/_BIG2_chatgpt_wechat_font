## 前端介绍

* 微信小程序数据请求源码: [点我进入](https://gitee.com/e0cia/chatgpt_wechat_houtai)

## 所需环境

uniapp
nodejs


## 搭建教学

* 首先前端源码下载下来，用idea源码编辑器打开,只需要修改配置文件中的请求api（request/request.js）,需要搭建好后端请求
* 然后直接启动项目即可(有任何我问题可以加入下方QQ群一起交流)
* 有问题的话群内有大佬协助,搞不好会有人帮你搭建谢谢
